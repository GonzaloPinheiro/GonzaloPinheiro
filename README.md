## 👋 Hola, soy Gonzalo

Soy un desarrollador de C# y .net junior.
He logrado terminar los estudios de DAM (desarrollo de aplicaciones multiplataforma) y aún continúo aprendiendo por mi cuenta tratando de mejorar con proyectos personales.

## 🚀 Qué hago ahora
Estoy centrado en seguir aprendiendo backend con .NET, creando proyectos pequeños o medianos para entender bien cómo funcionan las cosas por dentro: lógica, base de datos y estructura del código.
Personalmente prefiereo avanzar paso a paso, primero que funcione y cumpla su función para luego mejorarlo.

Las tecnologías sobre las que me estoy centrando en aprender las puedes leer en los readme de cada repositorio.

## 🧠 En qué me estoy enfocando
